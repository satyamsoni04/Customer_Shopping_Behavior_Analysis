# Customer_Shopping_Behavior_Analysis
Data Analytics project showcasing Customer Shopping Behavior Analysis using Python, SQL, and Power BI on 3,900 transactions (18 features). Python was used for data cleaning and EDA, SQL for business queries and analysis, and Power BI for interactive dashboards and actionable business insights.

# Customer Shopping Behavior Analysis — Data Analytics Project

## Overview

This project analyzes **Customer Shopping Behavior** using **Python, SQL, and Power BI** to uncover insights from **3,900 transactions (18 features)**. The goal is to identify customer trends, spending patterns, product performance, and business opportunities through data-driven analysis.

---

## Dataset

* **Rows:** 3,900 customer transactions
* **Features:** 18 columns
* **Key Fields:**

  * Customer ID
  * Age & Gender
  * Product Category
  * Purchase Amount
  * Discount Applied
  * Subscription Status
  * Shipping Type
  * Review Rating
